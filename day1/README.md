## 2. 测试是否安装成功

    
	在命令行输入 node -v 以及 npm -v  如果能输出版本号 则表示安装成功 
    


## 3. 什么是NodeJS？

    
	Node.js是一个让JavaScript运行在服务器端的开发平台，它让JavaScript的触角伸到了服务器端，可以与PHP、JSP、Python、Ruby平起平坐    
    

## 4. NodeJS的特点？

    
	单线程  -- 非阻塞I/O  -- 事件驱动
